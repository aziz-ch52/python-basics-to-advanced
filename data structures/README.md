## 📂 Project Architecture

### 🧠 Data Structures
This core directory covers the foundational building blocks for data manipulation.

* **[lists/](./data_structures/lists/):** Ordered, mutable collections. Essential for linear datasets and sequence processing.
* **[tuples/](./data_structures/tuples/):** Immutable, fixed-length records. Optimized for memory efficiency and data integrity.
* **[sets/](./data_structures/sets/):** Unordered, unique collections. Used for high-speed deduplication and set-math (SQL-like Joins).
* **[dictionaries/](./data_structures/dictionaries/):** Key-value mappings. The backbone of API interactions, JSON parsing, and fast-lookup metadata.
